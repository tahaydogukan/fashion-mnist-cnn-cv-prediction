# Fashion-MNIST Real-Time Classification (TensorFlow + OpenCV)

Bu proje, Fashion-MNIST veri seti ile bir görüntü sınıflandırma modeli eğitilmesini ve eğitilen modelin OpenCV üzerinden gerçek zamanlı olarak, yani webcam kullanılarak çalıştırılmasını amaçlar. Çalışma sürecinde iki farklı mimari test edilmiştir: yalnızca Dense katmanlarından oluşan temel bir model ve ardından daha yüksek doğruluk sağlayan CNN tabanlı bir model. CNN modeli başarıyla eğitildikten sonra .keras formatında kaydedilmiş ve canlı tahmin aşamasında kullanılmıştır.

#  Özellikler
  
Fashion-MNIST veri seti ile eğitim
Dense ve CNN model karşılaştırması
CNN ile yüksek doğruluklu sınıflandırma
OpenCV ile gerçek zamanlı webcam tahmini
Model kaydetme & model yükleme
10 sınıflı kıyafet sınıflandırma

#  Sonuç
  
Bu proje, Fashion-MNIST veri setiyle bir CNN modeli geliştirip bunu gerçek zamanlı webcam görüntülerinde kullanmayı hedefleyen modern bir derin öğrenme uygulamasıdır.
